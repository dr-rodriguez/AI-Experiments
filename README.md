# AI-Experiments

A repo with a variety of short prototypes using AI.   
For the most part, we use LangChain and Ollama.

Each directory contains a local venv environment. You can use `uv sync` to install dependencies and `uv run` to run code.

## Projects
- [x] Simple AI client using LangChain and Ollama
- [x] CLI chatbot
- [x] Streamlit app with AI chat
- [x] Agent with local tool
- [ ] Agent with external MCP tool
- [ ] Creating shared RAG with LangChain
- [ ] Agent with RAG
